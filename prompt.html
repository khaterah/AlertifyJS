Prompt Dialog

A prompt dialog is often used if you want the user to input a value. When a prompt dialog pops up, the user will have to click either "OK" or "Cancel" to proceed after entering an input value.
AlertifyJS

Prompt

Default value
OK
Cancel

Default usage

You create a prompt dialog by invoking alertify.prompt(...)

Calling alertify.prompt() without passing any parameter will return the prompt dialog instance. You can use this syntax to modify dialog settings before showing it. Also you can get the dialog instance (singletons only) by invoking alertify.dialog('prompt'); 


/*
 * @title {String or DOMElement} The dialog title.
 * @message {String or DOMElement} The dialog contents.
 * @value {String} The default input value. 
 * @onok {Function} Invoked when the user clicks OK button.
 * @oncancel {Function} Invoked when the user clicks Cancel button or closes the dialog.
 *
 * alertify.prompt(title, message, value, onok, oncancel);
 * 
 * overloads:
 *
 * alertify.prompt(message);
 * alertify.prompt(message, value);
 * alertify.prompt(message, value, onok);
 * alertify.prompt(message, value, onok, oncancel);
 *
 */  
alertify.prompt('This is a prompt dialog!', 'some value', 
    function(evt, value){ alertify.message('You entered: ' + value);}
);
Run

API reference

message { String or DOMElement }

Description: Sets the dialog contents

Default: undefined

 alertify.prompt().setting('message', 'This is a new message!').show();
Run

labels { Object }

Description: Sets the OK/Cancel button's labels

Default: {ok: alertify.defaults.glossary.ok, cancel: alertify.defaults.glossary.cancel}

 alertify.prompt('labels changed!').setting('labels', {ok:'Alright!', cancel:'Naa!'});
Run

reverseButtons { Boolean }

Description: Gets or sets a value indicating whether to reverse the dialog buttons order.

Default: false

 alertify.prompt('Buttons are reversed').setting('reverseButtons', true);
Run

onok { Function }

Description: Gets or sets a callback function to be invoked when the user clicks 'Ok' button.

Default: undefined

 alertify.prompt('a callback will be invoked on ok.').setting('onok', function(){ alertify.success('Ok');} );
Run

oncancel { Function }

Description: Gets or sets a callback function to be invoked when the user clicks 'Cancel' button or closes the dialog.

Default: undefined

 alertify.prompt('a callback will be invoked on cancel.').setting('oncancel', function(){ alertify.error('Cancel');} );
Run



You can use the .setting() method to set multiple values at once, and also works as a getter. 


//get the closable setting value.
var closable = alertify.dialog('prompt').setting('closable');
//grab the dialog instance and set multiple settings at once.
alertify.dialog('prompt')
  .setting({
    'labels':{ok:'Yup!', cancel:'Nop!'},
    'message': 'This dialog is : ' + (closable ? ' ' : ' not ') + 'closable.' ,
    'onok': function(){ alertify.success('Nice!');},
    'oncancel': function(){ alertify.error('Still nice!');}
  }).show();
Run

API reference (Common)

This refers to settings/methods that are common to all dialogs.

modal {Boolean}

Description: Sets the dialog display mode

Default: true

 alertify.prompt('Modal: false').setting('modal', false);
Run

movable {Boolean}

Description: Gets or sets a value indicating whether the dialog is movable.

Default: true

 alertify.prompt('Movable: false').setting('movable', false);
Run

resizable {Boolean}

Description: Gets or sets a value indicating whether the dialog is resizable.

Default: false

 alertify.prompt('Resizable: false').setting('resizable', true);
Run

closable {Boolean}

Description: Gets or sets a value indicating whether the Close button is displayed in the header of the dialog. (clicking the dimmer will also trigger a close command)

Default: true

 alertify.prompt('Closable: false').setting('closable', false);
Run

maximizable {Boolean}

Description: Gets or sets a value indicating whether the Maximize button is displayed in the header of the dialog.

Default: true

 alertify.prompt('Maximizable: false').setting('maximizable', false);
Run

pinnable {Boolean}

Description: Gets or sets a value indicating whether the Pin button is displayed in the header of the dialog (modeless only).

Default: true

 alertify.prompt('Modeless: false, Pinnable: false. Try scrolling the page!').setting({'pinnable': false, 'modal':true});
Run

transition {String}

Description: Gets or sets the transition effect to be used when showing/hiding the dialog

Default: alertify.defaults.transition:pulse

 alertify.dialog('prompt').setting({transition:'pulse',message: 'Transition effect: pulse'}).show();

onshow {Function}

Description: Gets or sets a function to invoke when the dialog is shown

Default: undefined

 alertify.prompt().setting({onshow:function(){ alertify.message('prompt was shown.')}});
 alertify.prompt('Demo');
Run

onclose {Function}

Description: Gets or sets a function to invoke when the dialog is closed

Default: undefined

 alertify.prompt('Demo').setting({onshow:null, onclose:function(){ alertify.message('prompt was closed.')}});
Run

overflow {Boolean}

Description: Gets or sets a value indicating whether the content overflow is managed by the dialog or not.

Default: true

 alertify.prompt().setting('overflow',false);

padding {Boolean}

Description: Gets or sets a value indicating whether the content padding is managed by the dialog or not.

Default: true

 alertify.prompt().setting('padding',false);

maximize ( ) chainable

Description: Maximizes the dialog.

Parameters: none

 alertify.prompt('This is a maximized dialog').maximize();
Run

restore ( ) chainable

Description: Restores the maximized dialog.

Parameters: none

 var div = document.createElement('div');
 div.innerHTML = 'This will be resored in 3 seconds';
 //show maximized dialog
 var prompt = alertify.prompt(div).maximize();
 //restore the dialog after 3 seconds
 var i = 3;
 var interval = setInterval(function(){
   if(--i == 0){
     clearInterval(interval);
     //restore the dialog
     prompt.restore().setting('message','Dialog restored.');
   }else{
     div.innerHTML = 'This will be resored in ' + i + ' seconds.';
   }
 }, 1000);
Run

pin ( ) chainable

Description: Pins a modeless dialog to the screen, preventing it from moving with body scroll.

Parameters: none

 alertify.prompt('This is a modeless dialog, pinned to the screen.').setting('modal', false).pin();
Run

unpin ( ) chainable

Description: Un-pins a modeless dialog from the screen, allowing it to move with body scroll.

Parameters: none

 alertify.prompt('This is a modeless dialog, not pinned to the screen.').setting('modal', false).unpin();
Run

isModal ( ) {Boolean}

Description: Gets a value that indicates whether the dialog is modal.

Parameters: none

 var isModal = alertify.prompt().isModal();

isMaximized ( ) {Boolean}

Description: Gets a value that indicates whether the dialog is maximized.

Parameters: none

 var isModal = alertify.prompt().isMaximized();

isPinned ( ) {Boolean}

Description: Gets a value that indicates whether the dialog is pinned.

Parameters: none

 var isModal = alertify.prompt().isPinned();
